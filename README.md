# Weather App

HTML / CSS / JavaScript で制作した、現在地取得対応の天気予報Webアプリです。  
都市名検索に加え、Geolocation API を利用した現在地取得機能を実装し、OpenWeatherMap API と連携して現在の天気と5日間予報を表示できるようにしました。  
時間帯（昼・夜）や天候に応じて背景や雰囲気が変化するUIを取り入れ、見やすさとデザイン性の両立を意識して制作したポートフォリオ作品です。

## 主な機能

- 都市名での天気検索
- 検索候補（サジェスト）表示
- 検索履歴の保存 / 削除
- 現在地取得（Geolocation API）
- 現在地の地名表示（reverse geocoding）
- 現在の天気表示
- 5日間天気予報表示
- 時間帯（昼・夜）による背景切り替え
- 天候に応じた背景 / 演出の切り替え
- ガラス風UIデザイン
- レスポンシブ対応

---

## 工夫した点

- 現在地取得時、英語表記の地名ではなく、分かりやすい市区町村名が表示されるよう reverse geocoding を調整
- 時間帯（昼・夜）と天候（晴れ・曇り・雨・雪など）を組み合わせて、背景や雰囲気が変化するUIを実装
- ガラス風デザインを取り入れ、視認性とデザイン性の両立を意識
- 5日間予報はPC・タブレット・スマートフォンで見やすいよう、レスポンシブ対応でレイアウトを調整
- バグ修正を行い、時間判定ロジックを整理して保守しやすい構成に改善
